## Baslayaq :)

Ilk once _Loading_ adli bir component yaradaq ve dinamik bir sekilde istifade rahatlassin

##### Loading.js

```js
import React from "react";

const Loading = ({ options }) => {
  return (
    <div
      className={`w-${options.width || 32} h-${options.height || 32} border-[${
        options.border
      }] rounded-full border-t-[${options.color || "purple"}] border-gray-400`}
    ></div>
  );
};
export default Loading;
```

### Istiafdesi

##### Index.js

```js
import React, { useState } from "react";
import Loading from "./components/loading";
const App = () => {
  const [loading, setLoading] = useState(false);
  const barOptions = {
    width: 40,
    height: 40,
    color: "red",
    border: 4,
  };
  return (
    <div>{loading ? <Loading options={barOptions} /> : <div>Loaded</div>}</div>
  );
};
export default App;
```
